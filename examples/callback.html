Drag a red square into one of the grey dropzones.

A custom callback has been written that changes the previous dropzone to purple and adds a sentence to the console.

Use the callback option to create your own custom callbacks. Your code will run only if dropAssure, dragAssure, and dropConditions are met.

Output

    
    <div class="draggable"></div>

    <div class="draggable"></div>

    <div class="droppable empty"></div>

    <div class="droppable empty"></div>

    <div class="droppable empty"></div>

    <script>

      (function(){

        $(document).ready(function(){

          $('.draggable').quickdroppable('draggable', {dragAddClass:'empty'});

          $('.droppable').quickdroppable('droppable', {dropRemoveClass: 'empty', dropCondition: '.empty'});

        });

      })(jQuery);

    </script>
