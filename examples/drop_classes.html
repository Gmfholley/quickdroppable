Drag the red square into either of the grey droppable zones.

When you start the drag event, it will add a class. When the drag event stops, it will remove it.

Use the dragRemoveClass and dragAddClass options to add/remove classes on a drag event. Use the dropRemoveClass and dropAddClass to add/remove classes on a drop event.

drag 
    
    <div id="draggable">drag</div>

    <div class="droppable"></div>

    <div class="droppable">

      <div id="first-child"></div>

    </div>

    

    <script>

      (function(){

        $(document).ready(function(){

          $('#draggable').quickdroppable('draggable', {dragRemoveClass: 'text-white red', dragAddClass: 'blue'});

          $('.droppable').quickdroppable('droppable', {dragRemoveClass: 'blue', dragAddClass: 'text-white red'});

        });

      })(jQuery);

    </script>
